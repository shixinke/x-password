# x-password

一个基于jQuery、用于显示和隐藏密码框和显示密码强度的插件

## 依赖

jQuery

## 使用手册

1、初始化参数

<table>
   <tr>
       <th>参数名</th>

       <th>含义</th>

   <tr>
       <td>strength</td>
       <td>boolean</td>

            level2:/^(?=.*?[A-Z])(?=.*?[a-z])(?=.*?[0-9]).{10,}$/,
            level3:/^(?=.*?[A-Z])(?=.*?[a-z])(?=.*?[0-9])(?=.*?[^\w\s]).{7,}$/,

$(selector).xPassword();
